# Kotesh Kumar Yelamati — Portfolio

Personal portfolio site hosted on GitHub Pages.

## Stack
- Plain HTML5, CSS3, Vanilla JS
- No frameworks, no build step
- Google Fonts (Inter + JetBrains Mono) via CDN

## Structure
```
kotesh-portfolio/
├── index.html
├── assets/
│   ├── css/style.css
│   ├── js/main.js
│   ├── kotesh-headshot.jpg      ← add your photo here
│   ├── KoteshKYelamati-Resume.pdf ← add your resume here
│   └── screenshots/
└── CLAUDE.md
```

## Deploy to GitHub Pages
1. Push this repo to `github.com/<username>/kotesh-portfolio`
2. Go to **Settings → Pages → Source: Deploy from branch → main / root**
3. Site will be live at `https://<username>.github.io/kotesh-portfolio/`

For a custom domain, add a `CNAME` file with your domain and configure DNS.

## Local preview
Open `index.html` directly in a browser — no server needed.
All paths are relative and GitHub Pages compatible.
